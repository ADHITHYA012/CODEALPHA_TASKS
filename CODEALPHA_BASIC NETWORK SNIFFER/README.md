# CodeAlpha Basic Network Sniffer

## Project Description

This project is a simple Network Sniffer developed using Python and Scapy. It captures network packets and displays useful information such as source IP, destination IP, protocol, and packet payload.

## Features

- Live packet capture
- Displays source IP address
- Displays destination IP address
- Displays protocol type
- Displays packet payload

## Technologies Used

- Python 3
- Scapy

## Installation

```bash
pip install scapy
```

## Run the Project

```bash
python network_sniffer.py
```

## Sample Output

```
===== Packet Captured =====
Source IP      : 192.168.1.5
Destination IP : 142.250.183.14
Protocol       : 6
Payload        : b'GET / HTTP/1.1'
```
